 ### Added
 - **pi-spinners**: Added whimsical thinking messages for turn starts and tool calls. Includes 400+ quotes across multiple categories (tech, scifi, fantasy, nonsense, etc.). #8
 - **pi-rfc-keywords**: Automatically uppercase RFC 2119 requirement keywords in prompts and commands. #9
 - **pi-remember**: Semantic long-term memory using local embeddings. Injects relevant context as hidden prompt messages. #10
 - **pi-hash**: Advanced content hashing with anchored read/apply tools for high-precision edits.
 - **pi-todos**: Interactive TUI-based task manager with checklist support and progress tracking.
