 # pi-hash

 Pi extension providing high-reliability file edits through line-based hashing.

 ## Features

 - `read`: Enhanced file reading with persistent line hashes.
 - `apply_patch`: Anchored diff application with spiral search and drift tracking.
 - Bash Guard: Protects against direct file modifications via shell commands.

 ## Usage

 The extension is activated by symlinking it into the `.pi/extensions` directory.
